動作例
    # reverse2 [];;
    - : 'a list = []
    # reverse2 [1;2;3];;
    - : int list = [3; 2; 1]
    # reverse [1;2;3];;
    - : int list = [3; 2; 1]
    # reverse [];;
    - : 'a list = []

考察
    reverse関数はアキュムレーターを用意することで
    リストの要素の最後の方からfi :: acmを再帰的に
    実行することで@を使わずに関数を実装している。
    reverse2関数の方ではeをtapleで持って、そこに
    リストとアキュムレーターを持たせることでreverse
    関数と同様の挙動を実現している。
