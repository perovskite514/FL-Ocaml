嶋村光太郎 05-211013

動作例
    # sum_to 10;;
    - : int = 55
    # is_prime 13;;
    - : bool = true
    # is_prime 12;;
    - : bool = false
    # gcd 10 5;;
    - : int = 5
    # gcd 5 7;;
    - : int = 1
考察
    fix関数は不動点コンビネータと呼ばれるものである。例えばsum_toでの適用例では
    sum_to = sum (fix sum) n と代入され、nがsumの第二引数に渡されることで、再帰構造が
    作られている。このように、不動点コンビネータによって無名再帰という自身を呼び出さない形での
    再帰が実現可能になる。
